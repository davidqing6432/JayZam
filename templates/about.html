Welcome to JayZam!

This website is made by David Qing! JayZam is a music recognition software that analyzes an audio file, and identifies what Jay-Z song it sounds most like. The project does so using machine learning; particularly classification! The website is built using HTML, CSS, JavaScript, and Python for the backend. To use this website, go to the main page by clicking on the logo in the top left, and click the record button. Then, use your microphone to supply some audio, and see what song your audio sounds most like!
